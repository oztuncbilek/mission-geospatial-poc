# Data Directory

This directory contains the spatial datasets and visual assets used in the HAT.tec Geospatial PoC.

## Contents
- `*.gpkg` (GeoPackage): Dummy mission data (flight paths, observations) generated by the Python pipeline. **Note:** These files are explicitly ignored by Git (`.gitignore`) to prevent uploading large binary spatial files to the repository.
- `readme_*.png`: Screenshots of the QGIS visualization and C++ terminal outputs used for the main documentation.
